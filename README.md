# Number Guessing Game

A simple and interactive command-line Number Guessing Game built with Node.js.
This project is based on the Number Guessing Game challenge from roadmap.sh.

The program randomly selects a number between 1 and 100, and the player must guess it within a limited number of attempts depending on the chosen difficulty.

# How to Run?
1. Ensure Node.js is installed on your machine.


2. Clone the repository with the following commands:
 
    _git clone https://github.com/azooony/Number-guessing-game_

    _cd Number-guessing-game_

3. Start the game:
 
    node taks.js
